# GearVN AI Marketing

Marketing assets và campaigns cho GearVN, powered by [ClaudeKit.cc](https://claudekit.cc).

## Giới thiệu ClaudeKit

**ClaudeKit** là bộ công cụ AI subagents và workflows giúp tăng cường khả năng của Claude Code cho teams phát triển phần mềm và marketing.

### Tính năng chính

- **108+ AI Skills** - Từ content creation, SEO, đến video production
- **95+ Slash Commands** - Tự động hóa các tác vụ lặp đi lặp lại
- **45 AI Agents** - 17 engineer agents + 28 marketing agents chuyên biệt
- **MCP Integrations** - GA4, Google Ads, Email automation, và nhiều hơn nữa

### Lợi ích

| Vấn đề thường gặp | ClaudeKit giải quyết |
|-------------------|----------------------|
| Token limits | Context management thông minh |
| Output không nhất quán | Workflows chuẩn hóa |
| Copy-paste prompts | Slash commands tự động |
| Setup phức tạp | Instant setup, không config |

### Packages

- **Engineer Kit** ($99) - Full-stack development, DevOps, code review
- **Marketing Kit** ($99) - 32 agents, 68 skills, production dashboard
- **Combo Bundle** ($149) - Cả hai kits
